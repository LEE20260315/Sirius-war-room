# Futures Tracker - Claude Design System

Redesigned with Claude Design Library tokens.

## Pages
- pool.html - 观察池 (品种跟踪)
- fundamental.html - 基本面 (五维评分)
- signal.html - 信号引擎 (三因子共振)
- trade.html - 模拟交易 (开仓/平仓)
- journal.html - 交易日志 (复盘记录)
- dashboard.html - 仪表盘 (资金曲线)
- settings.html - 设置 (系统参数)

## Architecture
- 7 standalone HTML pages + 5 shared JS modules
- Claude Design System tokens (brand #d97757, dark theme)
- Tailwind v4 CDN + @theme inline
- Google Fonts: Newsreader / Poppins / Lora / Geist Mono
